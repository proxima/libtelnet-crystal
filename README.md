# libtelnet

Crystal bindings for [libtelnet](https://github.com/seanmiddleditch/libtelnet)

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  libtelnet:
    github: proxima/libtelnet
```

Requires the user to have libtelnet installed locally.

## Usage

```crystal
require "libtelnet"
```
